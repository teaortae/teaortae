## About Me

I am an Android software engineer with 10+ years of experience building and operating real-world applications across franchise order services, store operation tools, Android TV signage, education apps, POS, MDM, and public-sector systems.

I like turning technology into maintainable products: stable releases, readable architecture, predictable state management, and user flows that keep working in the field.

## Focus

- Android Native development with Kotlin, Coroutine, Flow, StateFlow, and Jetpack Compose
- Service app architecture with Ktor, Koin, Hilt, DataStore, Room, and Firebase
- Kotlin Multiplatform and Compose Multiplatform client structure
- Desktop tools that connect OCR, Google Sheets, and operational workflows
